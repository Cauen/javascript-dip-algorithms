## Shortcuts

```
Tab > Open Config Panel
U > Upload Images
Click image > Set primary Image
DblClick Image > Set secundary image
= > Compare Images
CTRL+V > Paste Image to Board
CTRL+Space > Clear Board
CTRL+Click > Remove Image from Board
CTRL+RClick > Save Image
S > Search
```
